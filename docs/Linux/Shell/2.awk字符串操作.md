# awk

###### 如获取80端口使用情况

```sh
lsof -i:80 | awk '{print $2}'
```

###### 需要去过滤掉第一行，指令为：

```sh
lsof -i:80 | awk 'NR!=1{print $2}'
```
###### 每行按空格或TAB分割，输出文本中的1、4项
```sh
 # 格式化输出
 $ awk '{printf "%-8s %-10s\n",$1,$4}' log.txt
```

###### 忽略大小写

~ 表示模式开始。// 中是模式。

```sh
$ awk 'BEGIN{IGNORECASE=1} /this/' log.txt
---------------------------------------------
2 this is a test
This's a test
```

过滤完排序

```sh
cat log.log |awk '{print $12}' |sort |uniq
```
