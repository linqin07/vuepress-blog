# README

1GB=1024MB 、1MB=1024KB 、1KB=1024Byte 、1byte=8bit=1字节

​

byte  ->   8 bits  -->1字节
char  ->   16 bit  -->2字节
short ->   16 bits -->2字节
int   ->   32 bits -->4字节
float ->   32 bits -->4字节
long  ->   64 bits -->8字节

3亿个数字大小： 3亿 * 4字节 / 1024 / 1024 /1024 ≈ 1.11 G 内存
